About The Program

This is a control to simulate old game show displays. This is intended for use in Open Broadcaster Software for recording or streaming. The name "Eggcrate" is used by game show fans to describe this type of display. For more information, check out Jay Lewis' Display Field Guide.

Instructions

  • Double-click on the box to change the display. You can use the following characters:
    • 0123456789$.,
  • Press ENTER to see it in action.
  • Click on the Blink checkbox to make the display blink.
  • Click on the Countdown checkbox to start a countdown with the value as the number of seconds. It will automatically un-check when the countdown reaches zero.
  • Click on the Sound checkbox to make the display play a 'ding' sound whenever you change the value of the number.
  • Neat trick! To count up, set the value to zero, then check on the Countdown checkbox. It will think it's a negative number, but all you'll see is a positive number.
